# 交易纪律工作台 v0.1

## 打开方式

当前工作台已经启动在：

http://127.0.0.1:5173/

推荐用本地服务启动：

```bash
python3 server.py
```

然后打开：

http://127.0.0.1:5173/

如果直接打开本文件夹里的 `index.html`，基础记录功能可以使用，但行情同步和 K 线接口不可用。

## 当前版本包含

- 今日工作台：仓位、浮盈浮亏、买卖次数、未来事件、操作提醒。
- 持仓管理：成本价、当前价、仓位、止损、分批止盈、卖出/减仓记录，行情打开后自动刷新。
- 持仓图表：按通达信式结构拆成主图指标和副图指标，支持日线、周线、月线切换、缩放、止损/止盈计划位标注。
- 指标窗口：主图指标通过下拉选择，MA 周期可配置；副图通过下拉添加，支持成交量、成交额、MACD、KDJ、RSI，可关闭、拖动排序、双击放大。
- 自定义指标：已预留通达信公式录入入口，后续可接入公式解析和个性化指标渲染。
- 新建开仓：基础信息、仓位比例自动计算、技术检查、止盈止损、纪律检查；进入开仓前会根据当前交易计划做纪律劝阻。
- 复盘日志：开仓、持仓、减仓、止损、计划偏离、情绪记录、日复盘、周复盘。
- 交易计划：账户总资金、交易纪律周期、纪律总控、周计划、日计划、今日策略、今日任务、今日禁止事项。
- 事件：宏观经济数据、重大会议、重点公司事件；支持从官方公开日程在线更新。
- 事件日历：月历/年历视图，支持按事件类型和关联范围筛选；关联范围按国家/地区归类，中国范围包含 A 股、港股和台湾重点公司。
- 交易行为看板：账户盈亏曲线、纪律环境表现、违规开仓追踪、行为偏差信号、纠偏报告、资金风险、执行纪律、策略表现。
- 数据管理：导出备份、导入备份、清空数据。
- 行情接入：A 股实时价、日线/周线/月线 K 线、可配置 MA、成交量、成交额、MACD、KDJ、RSI。
- 行情刷新：打开持仓页后自动更新行情，也可以手动刷新当前价和 K 线。
- 今日工作台：交易纪律、市场状态、仓位纪律、开仓纪律优先展示，并同步交易纪律周期状态。
- 持仓管理：支持当前持仓、历史持仓、全部记录筛选，并可删除测试持仓。
- 交易行为看板：按纪律环境、违规开仓、行为偏差、资金风险、执行纪律、策略结构和当前风险项查看。

## 数据保存

当前版本的数据保存在浏览器本地。清除浏览器缓存或换浏览器前，请先在“数据”页面导出备份。

行情数据来自公开行情接口，当前版本用于个人复盘和纪律提醒，不作为投资建议。

## 下一阶段建议

1. 增加晚间自动扫描：同步行情、事件、纪律状态，生成次日操作提醒。
2. 增加交易回放：在 K 线图上标记买点、减仓点、清仓点和复盘备注。
3. 扩展事件来源：补充 CPI、非农、财报日历等更稳定的数据源。
4. 支持券商导出文件导入：用 Excel/CSV 对持仓和成交记录做核对。
5. 升级为本地数据库，并打包成可直接打开的桌面软件。
