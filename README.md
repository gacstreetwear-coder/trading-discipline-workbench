# 交易纪律工作台

个人股票交易记录、复盘、事件日历和交易行为纠偏工具。

## 使用方式

本地开发运行：

```bash
python3 server.py
```

打开：

```text
http://127.0.0.1:5173/
```

桌面版开发预览：

```bash
npm install
npm run dev
```

## 桌面安装包

项目已经配置 GitHub Actions 自动构建：

- macOS：生成 `.dmg` 和 `.zip`
- Windows：生成 `.exe` 安装包和 `.zip`

进入 GitHub 仓库后，在 `Actions` 页面运行 `Build desktop installers`，构建完成后在 `Artifacts` 下载安装包。

## 数据和账号

个人桌面版不需要账号，数据保存在本机。

网页登录版需要服务器、账号系统和云数据库，目前不是这个版本的默认形态。

券商账户不能无权限直接读取。当前支持券商 CSV 导入；真正直连需要券商开放 API 或用户授权。

## 主要能力

- 今日工作台
- 交易计划与纪律总控
- 持仓管理
- 新建开仓门禁
- 复盘日志
- 事件日历
- 交易行为看板
- 数据备份和券商 CSV 导入
- A 股行情、长历史 K 线、日线/周线/月线
