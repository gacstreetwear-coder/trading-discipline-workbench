# 交易纪律工作台桌面版

## 版本形态

当前桌面版是个人本机软件，不需要账号登录。

数据默认保存在用户自己的电脑里，适合你自己用，也适合发给少量朋友做内测。朋友之间的数据不会互通，也不会上传到服务器。

## 能自动连接的内容

- A 股行情和长历史 K 线：打开持仓后自动刷新，K 线支持日线、周线、月线和长历史缩放。
- 事件日历：可在线同步国家统计局、BEA、BLS CPI、非农、ADP 小非农、美国财政部、TIC、美联储 FOMC 等公开来源。
- 扫描报告：可以一键刷新行情、近期事件和操作提醒。

## 券商账户接入

券商账户不能无权限直接读取。

目前稳定可用的是“券商 CSV 导入”：从券商软件导出成交/对账 CSV，再导入工作台做复盘和对账。

真正的券商直连需要券商开放 API 或授权通道，例如部分海外券商、量化接口、仿真接口或机构接口。后续可以做成独立接入层，但不能绕过券商权限。

## 本机打包

先安装依赖：

```bash
npm install
python3 -m venv .venv-build
source .venv-build/bin/activate
python -m pip install --upgrade pip pyinstaller
```

开发预览：

```bash
npm run dev
```

打包 Mac 安装包：

```bash
source .venv-build/bin/activate
npm run dist:mac
```

打包 Windows 安装包：

```bash
source .venv-build/bin/activate
npm run dist:win
```

生成结果在 `release/` 文件夹。

## 自动生成 Mac 和 Windows 安装包

项目已经包含 GitHub Actions 自动构建配置：

`.github/workflows/build-desktop.yml`

把项目放到 GitHub 仓库后，手动运行 `Build desktop installers`，会分别在 macOS 和 Windows 环境生成安装包，并上传到构建产物里。

## 分发给朋友测试

发安装包即可：

- Mac：`.dmg`
- Windows：`.exe`

未签名版本第一次打开时，系统可能会提示“来自未知开发者”。正式对外分发前，需要购买 Apple Developer 账号和 Windows 代码签名证书。
